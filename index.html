Split your Tileset

How to split the tileset?
  1. Upload an Image:
    • Click the file input button to select an image file. Only valid image types (e.g., .png, .jpg) are accepted.
    • If no image is selected or the file is not an image, you will be prompted with an error message.
  2. Set Tile Dimensions:
    • Enter the desired width and height for each tile in the input fields labeled Tile Width and Tile Height.
  3. Click "Split Image":
    • Press the button labeled "Split Image" to start the process.
    • The image is split into tiles based on the width and height you provided. The process divides the image into a grid of smaller images.
  4. Download the Tiles:
    • Once the image is split, a ZIP file containing the individual tiles will be automatically generated and downloaded to your device.
